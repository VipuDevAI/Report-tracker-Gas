Manage Data Sheets

Create Class-wise Sheets

Creates the following sheets:

  • Students_11_A1 to Students_11_A12
  • Students_12_A1 to Students_12_A12
  • Teachers_Master
  • Class_Teachers

Each sheet will have proper headers with dropdown validations for Stream, Elective Subject, and Status.

Sync Students Data

Important: This will sync all students from class-wise sheets (Students_11_A1, etc.) to the main Students sheet.

Fill student data in class-wise sheets first, then click sync to update the main database.

Sync Teachers Data

Sync teachers from Teachers_Master sheet to the main Teachers sheet for login access.

Fill teacher data in Teachers_Master sheet first, then click sync to enable their login.
